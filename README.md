# Regional Sales Analysis (Python + Power BI)

## Project Overview

This project performs an end-to-end regional sales analysis to understand revenue patterns, profitability drivers, and customer behavior across multiple US states and regions.

Using Python for data exploration and Power BI for dashboard visualization, the analysis identifies top-performing markets, high-value customers, and key factors influencing sales performance.

The project demonstrates practical data analysis skills including data cleaning, exploratory data analysis (EDA), visualization, and business insight generation.

---

## Problem Statement

Businesses often struggle to understand which regions, customers, and products drive the most revenue and profitability.

The objective of this project is to analyze sales data to:

- Identify top-performing regions and states
- Understand revenue distribution across channels
- Analyze customer contribution to revenue
- Evaluate profit margins and pricing behavior
- Visualize insights through an interactive dashboard

---

## Dataset Description

The dataset contains transactional sales data including:

- Order number
- Customer name
- Product details
- Quantity
- Unit price
- Revenue
- Cost
- Profit
- Sales channel
- US region
- State

These features allow analysis of revenue performance, customer segmentation, and regional sales distribution.

---

## Key Insights

- California emerged as the highest revenue-generating state with the largest order volume.
- Revenue performance across states is primarily driven by transaction volume.
- A small group of customers contributes a significant share of total revenue.
- Profit margins vary significantly across pricing levels and product categories.
- Sales performance shows strong regional concentration.

---

## Power BI Dashboard

An interactive Power BI dashboard was created to visualize:

- Revenue by region
- State-level sales distribution
- Customer revenue contribution
- Order trends and sales performance

This dashboard enables quick identification of high-performing markets and customer segments.
